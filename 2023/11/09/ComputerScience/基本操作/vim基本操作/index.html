vim基本操作

发表于 2023-11-09

  • 词： w （下一个词）， b （词初）， e （词尾）

  • 行： 0 （行初）， ^ （第一个非空格字符）， $ （行尾）

  • 文件： gg （文件头）， G （文件尾）

  • 搜索: /{正则表达式}, n / N 用于导航匹配

  • x 删除字符（等同于 dl）

  • s 替换字符（等同于 xi）

  • 可视化模式 + 操作

    • 选中文字, d 删除 或者 c 改变

  • u 撤销, <C-r> 重做

  • y 复制 / “yank” （其他一些命令比如 d 也会复制）

  • p 粘贴

  • 更多值得学习的: 比如 ~ 改变字符的大小写

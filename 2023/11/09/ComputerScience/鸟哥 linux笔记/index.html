命令篇

  1. man 加/词 可以搜索你需要的词汇
  2. /usr/share/doc 有安装软件的信息
  3. shutdown -h how shutdown命令需要
  4. 除了使用cat看文件，还有tac（从最后一行开始显示），more（可以翻页，好用）
  5. alias 设置别名（如 ll 命令代表 ls -l）
  6. 压缩一整个目录，使用tar，压缩单个文件bzip2 gzip（-d解压）
    tar -cvf 目标名 文件名 压缩 ， tar -xvf解压
  7. grep用法 grep “word” filename

Bash Shell

  1. 格式
    #!/bin/bash
    value=”string”（不要有空格）
    echo $value

    执行使用sh filename.sh

  2. “%d”输出变量， ‘%d’输出%d（不转义）

  3. declare -afirx 定义变量为数组函数整数只读环境（默认为字符串）

  4. read value 读取变量

  5. $0 $1 $2 表示命令行参数sh file.sh $0 $1

  6. if then
    if [ “$yn” = “y” ] && [ 条件二 ]（注意空格）; then
    语句
    else
    语句
    fi 结束

  7. case
    case $value in
    one)
    语句
    value)
    语句
    exit 1
    esac

  8. for
    for (( i=1；条件2；i=i+1 ))
    do
    语句
    done

  9. while
    while [条件] || [条件]
    do
    语句
    done

  10. 调试脚本
    sh -nvx
    n 检查脚本错误
    v 执行脚本，并把脚本内容打印
    x 与v稍微不同(折叠显示)

1.0 GPT-SoVITS克隆音频

1.1 启动环境

  1. 准备wav格式音频
  2. 登录AutoDL子账号，并开启最新的GVS服务器（保持数据同步）
  3. WinSCP远程连接
  4. 开启虚拟机，开启指令：
  1. 打开网址，勾选UVR5-WebUI（分离人声、背景音）；退出，并打开新网址

1.2 处理音频

1.2.1 切割音频

  1. 文件路径：vocal.wav
  2. 输出目录 output/slicer_opt/tis/
  3. 开启语言切割
  4. 将输出目录填入ASR输入路径，等待任务完成

  2. 训练集文件目录 output/slicer_opt/tjs
  3. 一键三联，等待任务完成

vim基本操作

发表于 2023-11-09 更新于 2024-05-30

基本操作

  • 词：w（下一个词），b（词初），e（词尾）
  • 行：0（行初），^（第一个非空格字符），$（行尾）
  • 文件：gg（文件头），G（文件尾）
  • 搜索:/{正则表达式},n/N用于导航匹配
  • x删除字符（等同于dl）
  • s(substitute)替换字符（等同于xi）
    • 替换命令:{作用范围}s/{目标文本}/{替换文本}/{替换标志}
    • :%s/s_content/o_content/g 全局替换
  • 可视化模式 + 操作
    • 选中文字,d删除（剪切） 或者c改变
  • u撤销，<C-r>重做
  • y复制 / “yank” （其他一些命令比如d也会复制）
  • p粘贴
  • 更多值得学习的:
    • ~改变字符的大小写
    • 3w向前移动三个词
    • A（大写）可以迅速定位到行尾进行修改
  • %匹配括号

切换文件

-:e filename切换到filename文件
-:bn/bp切换到下/上个文件

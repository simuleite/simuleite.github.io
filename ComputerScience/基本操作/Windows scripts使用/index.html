发表于 2024-07-16

Task Scheduler

WIN+R, taskschd.msc
在这里可以设置定时任务、登录任务。
有些软件的开机自启动就设置在这里。比起组策略有更多的可选项（延迟启动，网络和电源要求等）

Group Policy Editor

WIN+R, gpedit.msc
在这里可以设置一些开关机时执行的任务

静默运行脚本

Task Scheduler没有办法隐藏窗口
在Task Scheduler运行该vbs脚本，目标脚本路径作为参数即可实现静默运行。
